## Descripción

Proyecto para el repaso con proyectos cortos para realizar en distintas pantallas.

## Instrucciones

Clonar el repositorio:

```
    git clone https://github.com/zeosjb/Ejercicios_Frontend.git
```

Una vez clonado el repositorio realizar:

```
    npm install
```

Dentro de la carpeta app/(tabs) se encuentran las instrucciones para construir cada una de las pantallas pedidas, se puede utilizar cualquier librería de estilos o estilos creados desde 0.

Posteriormente crear un repositorio con sus soluciones y subirlos al campus virtual.
